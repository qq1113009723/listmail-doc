    SDKs and client libraries¶

    A list of 3rd party client libraries and SDKs that have been written for stmails APIs.

    Note

    The list is community sourced. They have not been verified and are not officially supported.

    • WordPress - WooCommerce plugin integration for stmails
    • stmails — Python API client
    • stmails-api — Python API client
    • frappe_stmails — Frappe framework integration for stmails
    • auto-newsletter-stmails — Ghost CMS integration
    • stmails-newsletter - RSS to stmails integration for email newsletters
    • stmails-crysctal — Crystal lang API client
    • terraform-provider-stmails — Manage stmails templates in Terraform
    • stmails-php-client — PHP API client
    • php-stmails — PHP API client
    • go-stmails — Go API client
    • stmails-nodejs-api — NodeJS API client
      • stmails-laravel — Laravel API Client
